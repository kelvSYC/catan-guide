Welcome to the _Catan Scenario and Variant Guide_ Live Edition, the official continuation of the _Catan Scenario and Variant Guide_, last seen in 2017.

This site is currently a work in progress, as much of the content is being rewritten for this new format.

## The Base Game

- [The First Island]({% link products/base-game.md %}) - Everything you need to know about the base game, and all its forms
- [The Second Islands]({% link products/base-game.md %}#scenarios) - Attempts to expand on the base game, predating the expansions

**Fixed-board versions or versions with simplified setup**:
- _Simply Catan_ - 2006 simplified setup game
- _Die Siedler von Catan: Einsteiger-Variante_ - 2007 simplified setup game

**Re-themed versions of the base game**:
- _The Settlers of Zarahemla_ - 2003 re-theme of the base game, with unique rules
- _Rockman.EXE Catan_ - 2005 re-theme of the base game, includes rules for a unique scenario
- _Star Trek Catan_ - 2012 re-theme of the base game and [_Helpers of Catan_]({% link products/helpers.md %})
  - _Star Trek Catan: Federation Space_ - 2013 add-on that adds a themed _Catan Geographies_ series scenario.
- _Catan Collectors Edition: Ancient Egypt_ - 2014 standalone game with a unique scenario
- _A Game of Thrones Catan: Brotherhood of the Watch_ - 2017 standalone game with a modified version of the base game as a variant
- [_Catan Core + China Map_]({% link products/core-china.md %}) - 2019 Big Box release that includes the base game and standalone scenario

**Standalone games with base game mechanics**:
- _Die Siedler von Nürnberg_ - 1999 standalone game
- _The Settlers of Canaan_ - 2002 standalone game with biblical theme
- _Die Siedler von Canaan_ - 2005 standalone game, similar to but with differences from _The Settlers of Canaan_
- _Catan Geographies: Germany_ - 2008 _Catan Geographies_ series standalone game
- _Baden-Württemberg Catan_ - 2012 _Catan Geographies_ series standalone game
- _Wien Catan_ - 2013 _Catan Geographies_ series standalone game

## Past Expansions

- [_Historische Szenarien I_]({% link products/historische-szenarien-i.md %}) - 1998 collection of two scenarios on a fixed board inspired by history. Never released in English, though German sets were sold with an attached English translation.
- [_Das Buch zum Spielen_]({% link products/das-buch-zum-spielen.md %}) - A book of scenarios and variants released in 2000. Never released in English, though gameplay-related portions of the book were released with an attached English translation.
- [_Historische Szenarien II_]({% link products/historische-szenarien-ii.md %}) - 2001 collection of two scenarios on a fixed board inspired by history. Never released in English, though German sets were sold with an attached English translation.
- [_Atlantis_]({% link products/atlantis.md %}) - A 2005 collection of scenarios taken from [_Das Buch zum Spielen_]({% link products/das-buch-zum-spielen.md %}), celebrating the 10th anniversary of the base game. Never released in English.
- [_Die Kolonien_]({% link products/die-kolonien.md %}) - A 2007 mini-expansion released for the 10th anniversary of _Seafarers_. Never released in English.

## Additional Scenarios

- [_Helpers of Catan_]({% link products/helpers.md %}) - 2010 standalone scenario

- [_Santa Claus_]({% link products/santa-claus.md %}) - 2015 holiday promotion scenario. Released under many different names for many different occasions.
- [_Hawaii_]({% link products/hawaii.md %}) - 2017 scenario originally part of the _Catan Geographies_ series, requiring _Seafarers_ to play, now recognized as its own scenario.
- [_Crop Trust_]({% link products/crop-trust.md %}) - 2018 standalone scenario
- [_#WeStayHome_]({% link products/westayhome.md %}) - 2020 Free print-and-play scenario
- [_Helpers_]({% link products/helpers.md %}) - 2022 revision of 2010's "Helpers of Catan"
- [_Soccer Fever_]({% link products/soccer-fever.md %}) - 2023 standalone scenario
- _Mowlin, the Benevolent Ram_ - 2025 reward for completing the escape room game _Exit: Adventures on Catan_
- _Ein Herz für Den Räuber!_ - 2025 standalone scenario

- _Catan Geographies_ series - Promotional scenarios played in the style of the spinoff game _Catan Geographies: Germany_, but with a standard _Catan_ set.

- _Catan-News_ scenarios - Scenarios included in the _Catan-News_ German newsletter.
- [_Der Schokoladenmarkt_]({% link products/der-schokoladenmarkt.md %}) - 2010 promotional collaboration, never released in English.
