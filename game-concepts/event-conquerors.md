In the scenarios of the [_Legend of the Conquerors_][conquerors] campaign, **events** occur whenever the barbarian ship on the [Chapter Board][chapter-board] advances to a specifically marked space. Many of these events are common to multiple scenarios.

- **Barbarian Attack**: These events resolve as they do in [_Cities & Knights_][cities-knights].
- **The Conquerors Land**: This event is associated with a strength rating. When this event resolves, a [Conqueror][conqueror] with the specified strength rating is placed on every hex with a [Landing Marker][landing-marker].
  - Note that due to the layout of the [barbarian track][chapter-board], there should not be a case where [Conquerors][conqueror] are to be placed on hexes where a [Conqueror][conqueror] is already present.
- **The Conquerors Advance**: When this event resolves, all of the [Conquerors][conqueror] on the board move. This event is associated with either "red side first" or "green side first", which determines the order in which the conquerors are to move.
- **Wyler's Help**: When this event resolves, each player, starting with the player to move and proceeding in turn order, may purchase up to two [Battle Progress Cards][battle-progress-card]. These cards are drawn from the supply, and are kept until used.
  - Any currency paid towards the purchase of [Battle Progress Cards][battle-progress-card] is returned to the supply.
- **Support from the South**: This event is associated either with [amber][amber-token] or [wine][wine-token]. When this event resolves, each player, starting with the player to move and proceeding in turn order, takes one token of the specified type from the supply.
  - If the supply is out of tokens for that particular type, then the remaining players receive no tokens.
- **Support from the East***: This event only happens in [Free All of Catan!][free-all-of-catan]. When this event resolves, each player, starting with the player to move and proceeding in turn order, may place one road on the board at no cost, so long as the road is on or beyond a line of demarcation on the board, indicated on the setup charts.
  - Players may not build a road if they have no road pieces in their supply, or if their only road placement options are behind the line of demarcation.
- **Charge of the Conquerors**: This event only happens in [Free All of Catan!][free-all-of-catan]. When this event resolves, all knights on intersections adjacent to [Forts][fort] are removed from the board.
